# pixel-map
Pixel Map is a DApp which powers a pixel board controlled by transactions on the Etheruem Blockchain

### Install

```
$ npm install
```

### Testrpc

```
$ testrpc
```
To start a local etheruem testing client

### Compilitation and Migratration (truffle)

```
$ truffle compile
$ truffle migrate

```

### Run

```
$ npm run dev
```
